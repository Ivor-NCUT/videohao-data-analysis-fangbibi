# 视频号数据分析@方比比

方比比视频号内容复盘 skill，用于把视频号后台数据、单条视频表现、多条内容对比或运营观察，分析成下一轮可执行的内容迭代动作。

核心框架：

```text
事实 -> 归因 -> 迭代
```

## 适用场景

- 分析视频号后台截图或数据表。
- 复盘单条视频的数据表现。
- 拆解转发、爱心、评论、播放、完播、涨粉等异常值。
- 判断一个选题是否命中目标人群的底层渴望。
- 把内容数据转成下一条视频的选题、标题、开头、结构和评论区运营建议。

## 分析维度

### 事实

先识别异常值：什么数据高了、低了、反常了，哪些组合值得重点看。

### 归因

围绕用户动机解释异常：

- 转发动机：用户为什么愿意把这条视频发给别人。
- 爱心动机：用户为什么愿意轻量表达喜欢或认同。
- 评论动机：用户为什么愿意多花一步打字。
- 选题命中：这条内容命中了人群的哪种底层渴望。

### 迭代

把判断落到下一步动作：

- 选题怎么延展。
- 标题/封面怎么强化。
- 开头 3 秒怎么设计。
- 叙事结构怎么调整。
- 评论区怎么运营。
- 是否值得系列化。
